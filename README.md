This app is a sleep tracker for Cognitive Behavioral Therapy for Insomnia (CBT-I).
Data is stored in a Supabase database, graphs are generated using Plotly.
User can enter, edit, and delete sleep data for each day.
History shows table of recent entries, and Dashboard shows graphs of key CBT-I metrics like Sleep Efficiency.
